Program Description

What This Credential Represents

The Kith Climate 6-Week Cohort is a live, online program where climate professionals master Claude Code and build working AI-powered deliverables for real sustainability problems. Over 6 weeks, participants progress from a client pitch through to a capstone disclosure narrative, producing six client-ready deliverables along the way.

This is not a lecture-based certificate program. Graduates have demonstrated the ability to build professional-grade climate tools and analysis using Claude Code — from data audits through a client-facing application and a disclosure-ready narrative.

The program requires 5–7 hours per week of active participation including live workshops, office hours, demo presentations, and independent building. Maximum cohort size is 50 professionals.

Curriculum

Week-by-Week Breakdown

Each week pairs a real climate deliverable with the underlying Claude Code skill that produced it, following a build–apply–present cadence: Monday live workshop, mid-week refinement and office hours, Friday demo session.

Week 1
Client Pitch
Proposal & Claude Code setup
Configure Claude Code for climate work — CLAUDE.md, folder structure, models and permissions — and produce a client pitch that scopes and frames the engagement.
Week 2
Data Audit
Data cleaning, ingestion & analysis
Ingest, clean and analyse real climate datasets; produce a data-quality assessment surfacing gaps, completeness and prioritised recommendations.
Week 3
Carbon Inventory & Dashboard
Carbon
Build an interactive emissions inventory across Scope 1, 2 and 3 with a live dashboard, traceable back to source activity data.
Week 4
Materiality Assessment
Materiality
Create a materiality matrix and a prioritised reduction roadmap — strategic deliverables that move from data to decisions.
Week 5
Client-Facing App
App-building
Turn the analysis into a working, client-ready application — an interactive tool a sustainability team could actually use.
Week 6
Capstone Disclosure
Assurance
Synthesise the body of work into a disclosure-ready capstone, with the QA, controls and audit trail needed to make AI-produced output trustworthy.
Completion Requirements

What Graduates Demonstrated

To earn this credential, the graduate met all of the following requirements:

  • Attended live sessions throughout the 6-week program
    Monday workshops, Wednesday office hours, and Friday demo sessions.
  • Built six client-ready deliverables with Claude Code
    A client pitch, data audit memo, carbon inventory dashboard, materiality matrix, client-facing app, and a capstone disclosure narrative.
  • Presented and demoed their work to peers
    Shared deliverables in Friday demo sessions, received and incorporated feedback, and explained methodology and findings.
  • Demonstrated proficiency with Claude Code for climate work
    Configured a CLAUDE.md, integrated live data, produced analysis and visualisation, built a client-facing app, and completed an end-to-end assured build.
Climate Topics

Climate Topics Covered

Across the deliverables, graduates worked hands-on with the core of the sustainability consulting stack:

Claude Code Skills

Claude Code Capabilities

Beyond the climate work, graduates developed a working command of Claude Code across the full build stack:

  • Foundations & operating Claude Code — Claude.ai vs Claude Code, model selection & effort, permission modes & safe data handling, CLAUDE.md & project architecture, reference-file creation & management.
  • Building deliverables — interactive HTML deliverables, interactive app design, Supabase integration, external & Anthropic API integration, report generation & document automation.
  • Data & analysis — Claude Code–assisted data analysis, document ingestion & classification at scale.
  • Trust & assurance — assurance & controls integration, QA system design, audit trails & traceability, deterministic-vs-probabilistic design.
  • Working method — iterative querying & prompt craft.
