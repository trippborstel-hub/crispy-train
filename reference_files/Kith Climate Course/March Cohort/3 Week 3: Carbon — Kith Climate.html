Three Scopes
Where the Northwood data sits today
Scope 1
Direct emissions — natural gas, diesel, propane, refrigerants. What you burn and release on-site.
Submitted ✓
Scope 2
Purchased electricity — emissions from the grid delivering power to your facilities.
Submitted ✓
Scope 3
Value chain — suppliers, commuting, business travel, customer use, everything else. Often 70–90% of total emissions.
Framework only
The submitted CSV covers Scope 1 and 2. Scope 3 is not zero — it's unquantified.
How the Math Works
It's arithmetic, not a black box
Consumption
kWh, therms, gallons — what the portco reported in the CSV
×
Emission Factor
EPA EF 2023 (Scope 1) · EPA eGRID 2022 by state (Scope 2)
=
mt CO2e
Metric tons of CO2-equivalent — the standard unit
Scope 1 example
Natural gas: 1 therm × 0.00531 mt CO2e/therm → 5.31 kg CO2e
Scope 2 example
Illinois electricity: 1,000 kWh × 0.000371 mt CO2e/kWh → 0.371 mt CO2e
The factors are in the climate data collection protocol. Claude Code applies them.
How AI Does the Math
Guessed or computed — the answer looks the same either way
Predicts the number
The model answers from pattern, like autocomplete. Fast, no code — fine for a rough estimate or a sanity check. But a wrong number looks exactly as confident as a right one.
Trusting: the model's memory
Writes & runs code
The model writes Python and runs it. 1,000 kWh × 0.000371 now truly evaluates to 0.371 mt — a real product, not a plausible guess. The logic can be wrong; the arithmetic can't.
Trusting: the model's logic + Python's arithmetic
Code you own
The Claude Code path. Same exact computation, but the script is yours to read, test, and rerun. The only mode that yields a number you can defend to an auditor.
Trusting: your own reading of the code
Code guarantees the arithmetic — not that you picked the right factor or unit (kWh vs. MWh). That part's still on you.
Data Quality Changes the Answer
A
Utility Bills
Actual consumption from invoices. No estimation. You can trust this number.
~60% of Northwood's facilities
B
Accounting Estimates
Derived from financial records — e.g., gas expense ÷ avg rate/therm. Methodology-dependent.
~20% of facilities
C
Landlord / Proxy
Building averages, EPA benchmarks, sq ft estimates. Last resort. Must be flagged.
~20% of facilities
A dashboard built 20% on estimates should say so. Data quality is a deliverable, not a footnote.
Intensity Is the Comparable
Sterling emits 5,802 mt CO2e. Keystone emits 725 mt. Does Sterling have a worse climate profile?
mt CO2e per $M Revenue
Normalizes for size. A $225M BPO firm and a $45M behavioral health network aren't the same scale. Revenue intensity makes them comparable.
Revenue is in Supabase.
mt CO2e per Employee
Normalizes for workforce size. Useful for services businesses where headcount drives energy consumption more than revenue.
Headcount is in Supabase.
Intensity is how Sarah compares a manufacturer to a clinic. Absolute emissions can't do that.

  • Patriot Staffing's 3,200 temp workers commute to client sites. That's Category 7 — not in any submission.
  • Harvest Ridge Foods buys agricultural ingredients. That's Category 1 — typically 80%+ of a food company's footprint.
